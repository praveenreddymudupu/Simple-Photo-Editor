# Simple-Photo-Editor

# Photo Editor using OpenCV and Streamlit

## Project Description

This project is a web-based Photo Editor built using **OpenCV** and **Streamlit**. It allows users to upload an image and apply multiple image processing operations interactively using sliders and buttons.

The application provides real-time image editing with an easy-to-use interface and allows users to download the edited image.

---

## Features

### Basic Features

* Upload an image
* Resize image
* Adjust brightness
* Adjust contrast
* Convert image to grayscale
* Apply blur effect
* Apply warm filter
* Portrait-style background blur
* Sharpen image
* Download edited image

### Extra Features

* Edge detection
* Sketch effect
* Cartoon effect
* Rotate image
* Flip image
* Sepia filter

## Installation

Clone the repository

```
git clone <repository-url>
```

Move into the project

```
cd Photo-Editor-OpenCV-Streamlit
```

Install dependencies

```
pip install -r requirements.txt
```

Run the application

```
streamlit run app.py
```
